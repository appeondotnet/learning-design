# 代理模式
为另一个对象提供一个替身或占位符以控制对这个对象的访问。

## 常用的代理
- 远程过程调用 RPC，具体例子 [Grpc](https://docs.microsoft.com/zh-cn/aspnet/core/grpc/)
- 跨进程通信 IPC

## 举例
- [WebApiClient](https://github.com/dotnetcore/WebApiClient#%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8)
- 新框架中的 IPC 框架
